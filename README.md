## Configuração

### Database (.env)

Altere com os dados de acesso ao seu MySql

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=book_search_google_books
DB_USERNAME=root
DB_PASSWORD=
```

### Migrate

```
php artisan migrate
```

## Deploy

### Desenvolvimento

```
npm run dev
```
